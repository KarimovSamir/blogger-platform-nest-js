Контроллер — принимает запросы.
Сервис — управляет логикой (хэширует пароли, командует).
Сущность (Entity) — чертеж данных с умными методами (createInstance, makeDeleted).
Репозиторий — пишет данные в БД.
Query-репозиторий — только читает из БД для отдачи клиенту.
DTO — валидирует входящие данные и формирует исходящие.
